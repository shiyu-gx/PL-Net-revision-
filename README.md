# PL-Net

This repository provides the code supporting the main findings of the paper:

**PL-Net: A physically constrained dual-stream framework for earthwork segmentation and volume estimation from UAV photogrammetry under sparse annotation**

## Overview

PL-Net is a physically constrained dual-stream framework for earthwork-region segmentation and subsequent volume estimation from UAV photogrammetry under sparse annotation. The framework integrates elevation-aware modeling, ΔDEM-guided shape priors, cross-domain spatial adaptive fusion, and three-level collaborative supervision.

## Contents

- Training and inference code for PL-Net
- Core model implementation
- Configuration files and scripts
- Example usage instructions

## Data Availability

The code supporting the main findings of this study is publicly available in this repository.

The datasets generated and/or analysed during the current study are not publicly available due to site confidentiality, third-party permission restrictions, and the presence of sensitive location information in the raw UAV imagery, but are available from the corresponding author on reasonable request.
